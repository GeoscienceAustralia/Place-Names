Description

Place Names is an RDF/OWL Ontology to define the structure and relationship of the names of geospatial features, typically collected in a Gazetteer.

A place-name is a text string that

  • is coined by an agent (person or organization)
  • is authorized by a jurisidiction (national, state or local government)
  • is a member item in a gazetteer
  • has lifecycle information (status, validity dates (TBD), formal identifier)
  • denotes a natural or constructed geospatial feature with a specified classification (inhabited place, administrative area, mountain, school, station, park ...)

This ontology uses classes and properties from existing vocabularies, in particular

  • GeoSPARQL
  • SKOS
  • Dublin Core
